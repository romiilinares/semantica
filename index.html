elementos semanticos

etiquetas para destacar texto

u
Muestra el texto subrayado.
ejemplo:river
i
Muestra el texto marcado con un estilo en cursiva o italica.
ejemplo: river el mejor
abbr
b
indica que el texto debe ser representado con una variable bold, o negrita, de la tipografía que se esté usando
ejemplo:chau
br
Es útil para escribir un poema o una dirección, donde la división de las líneas es significante.
ejemplo:
holaa
cite
Marca una referencia a una fuente, o el autor de un texto citado.
ejemplo: el mas grande
strong
El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto.
ejemplo: holaa
s
Muestra el texto tachado con una linea horizontal
ejemplo:romina
sub
define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice.
ejemplo:linares
sup
define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice.
ejemplo:como estas
span
Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea
ejemplo:hola profe
time
representa un periodo específico en el tiempo
ejemplo:buenas tardes
code
Es el apropiado para marcar el código de un programa
ejemplo:chauu
a
crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra UR
ejemplo:chau porfin
